# shtWay
Atalhos para algumas funções fundamentais do Javascript
<br>
<br>
### Instalação

```html
<script src="shtWay.js" type="text/javascript"></script>
```

### Exemplo de uso

```javascript

var obj = ce("div");

obj.sty({
  width: "100px",
  height: "100px",
  backgroundColor: "#000000",
  color: #ffffff
  // mais estilo
});

obj.atr({
  id: "box",
  className: "listBox"
});

document.body.ape(obj);

ide("box").ape(txt("exemplo shtWay"));

```
